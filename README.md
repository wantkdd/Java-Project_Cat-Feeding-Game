## 📖 게임 소개

Cat Feeding Game은 Java Swing으로 제작된 타이핑 연습 게임입니다.
화면에 떨어지는 물고기에 적힌 단어를 빠르게 입력하여 고양이에게 밥을 먹여주세요!
단어를 놓치면 고양이가 배고파하고, 생명이 줄어듭니다. 생명을 모두 잃으면 게임이 종료됩니다.

## ✨ 주요 기능

### 🎯 게임 메커니즘
- **타이핑 게임**: 떨어지는 물고기의 단어를 입력하여 제거
- **생명 시스템**: 5개의 하트로 시작, 단어를 놓치면 감소
- **난이도 선택**: Easy, Normal, Hard 3가지 난이도 제공
- **점수 시스템**: 난이도별 차등 점수 (1배/2배/3배)
- **순위 시스템**: 최고 점수 TOP 10 저장

### 🎁 특수 아이템
| 아이템 | 설명 | 효과 |
|--------|------|------|
| 🐟 **일반 물고기** | 기본 아이템 | 난이도별 점수 획득 (1~3점) |
| 🐠 **큰 물고기** | 고득점 아이템 | +5점, 코인 효과음 재생 |
| ❤️ **하트** | 생명 회복 아이템 | 생명 +1 (최대 5개) |
| ⏸️ **타임스탑** | 시간 정지 아이템 | 모든 물고기 2초간 정지 |

### 🐈 고양이 상태 변화
- **평온 상태**: "밥 달라냥" - 게임 시작 시
- **배고픈 상태**: "밥 안 주냥" - 단어를 놓쳤을 때
- **만족 상태**: "맛있다냥" - 단어를 맞췄을 때

## 🎮 게임 방법

### 시작하기
1. 게임을 실행하면 **시작 화면**이 나타납니다
2. **난이도 선택**: Easy / Normal / Hard 중 선택
3. **Start 버튼** 클릭으로 게임 시작

### 게임 플레이
1. 화면 위에서 물고기가 단어와 함께 떨어집니다
2. 하단 입력창에 **단어를 정확히 입력**하고 Enter를 누릅니다
3. 단어가 맞으면 물고기가 사라지고 점수 획득!
4. 단어를 놓치면 생명(하트)이 감소합니다
5. 생명이 0이 되면 게임 종료

### 단어 추가
- 시작 화면에서 **단어 추가 기능** 사용 가능
- 입력창에 단어를 입력하고 추가 버튼 클릭
- 추가된 단어는 게임에 즉시 반영됩니다

## 🎯 난이도별 상세 정보

| 난이도 | 낙하 속도 | 점수 배율 | 일반 물고기 | 큰 물고기 | 하트 | 타임스탑 |
|--------|----------|----------|------------|----------|------|----------|
| **Easy** | 300ms | 1배 | 89% | 5% | 5% | 1% |
| **Normal** | 200ms | 2배 | 77% | 10% | 10% | 3% |
| **Hard** | 100ms | 3배 | 65% | 20% | 10% | 5% |

> 💡 **팁**: Hard 난이도에서는 특수 아이템 출현 확률이 높아집니다!

## 🚀 설치 및 실행 방법

### 필요 사항
- **Java JDK 8 이상** 설치
- 게임 리소스 파일:
  - 이미지: `peacefulCat.jpg`, `hungryCat.jpg`, `fullCat.jpg`, `fish.png`, `bigFish.png`, `heart.png`, `timeStop.png`, 배경 이미지 등
  - 음악: `lobbyBgm.wav`, `easyBgm.wav`, `normalBgm.wav`, `hardBgm.wav`, `coinBgm.wav`
  - 데이터: `words.txt` (단어 목록), `rank.txt` (순위 기록)

### 컴파일 및 실행
```bash
# 소스 컴파일
cd src
javac *.java

# 게임 실행
java GameFrame
```
